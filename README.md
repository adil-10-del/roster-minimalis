# roster-minimalis
Perusahaan di bidang konstruksi yang menyediakan interior bangunan, memproduksi roster beton, bata ekspos, ornamen dinding, dan elemen dekoratif lainnya dengan kualitas tinggi. Mengutamakan desain modern, kekuatan material, serta pelayanan terbaik untuk memenuhi kebutuhan hunian, komersial, dan proyek konstruksi di berbagai skala.
